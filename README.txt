Интерфейс приложения состоит из:
	Меню;
	Поле ввода текса пользователем;
	Поле вывода обработанного текста;
	Кнопка запуска преобразований;
	Поле ввода шага шифрования.
В меню пользователь может:
	Выбрать файл для редактирования;
	Сохранить результат обработанного текста в файл;
	Выбрать, шифровать или дишифровать полученную информацию;
	Выбрать направление шифрования/дишифрования;
	Выбрать алгоритм шифрования (шифровать буквы и цифры по отдельности или вместе).
Дополнительный функционал:
	При импорте, текст из файла помещается в ячейку для ввода текста. Пользователь имеет право дополнять текст из файла своей информацией;
	При импорте вся ранее введёная информация в окне ввода будет стёрта и заменена тексом из файла. Если файл не выбран, или выбран не верный файл, поле просто станет пустым;
	При импорте файла не формата .txt будет выдано сообщение об ошибке;
	При экспорте без преобразований, будет создан пустой файл. Пользователь имеет право создать файл с "зашифрованной" пустой информацией;
	При вводе в поле для шага отрицательного числа или буквы, пользователь будет оповещён об ошибке, операция преобразования не будет произведена;
	При отмене импорта или экспорта пользователь будет оповещён об отсутствии файла.
	Программа шифрует только цифры и буквы русского алфавита. Остальные символы останутся без изменений;
	Поле вывода не будет меняться до тех пор, пока не будет нажата кнопка "Преобразовать".
